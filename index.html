Features of JS:

JavaScript exicution thred is a single thread which means it can handle a piece of code at a time. For this reason, it is called single thread programming language.It is an interpreted language, this means we do not have to compile the JavaScript source code before sending it to the browser. An interpreter in the browser reads over the JavaScript code, interprets each line, and runs it. It runs inside an Internet browser. Every browser provides a JavaScript engine that runs the JavaScript code which means JavaScript engine convert the javascript code into machine language.The source code is passed through a program called a compiler, which translates it into bytecode that the machine understands and can execute. JavaScript is using a callback queue.The Engine consists of two main components: Memory Heap which is used for memory management and Call Stack which is used to store the function in the form of stack(which function is first pushed to the stack is poped from the stack to the end and which function is pushed to the end of the stack is first poped from the stack) when we call the function in JS. It works both in synchronous and asynchronous way.
